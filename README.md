# stdlib-utils

A collection of utility functions, types and macros for the [**Swift Standard Library**](https://developer.apple.com/documentation/swift).

Currently these utilities are split into 3 sections:
- [**TypeUtils**](https://floormatgen.github.io/stdlib-utils/documentation/typeutils/)
- [**ConcurrencyUtils**](https://floormatgen.github.io/stdlib-utils/documentation/concurrencyutils)
- [**ObservationUtils**](https://floormatgen.github.io/stdlib-utils/documentation/observationutils/)
